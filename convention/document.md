# 文档编写规范

## 基本原则

1. 结构化优先：表格、列表优于长段落
2. 状态显式：任务、文件标注状态
3. 路由分离：索引存路由，内容存具体文件

## 状态符号

| 符号 | 含义 |
|------|------|
| ✅ | 完成 |
| 🚧 | 进行中 |
| ⏸️ | 暂停 |
| ❌ | 取消 |

## 文件命名

| 类型 | 规则 | 示例 |
|------|------|------|
| 索引 | README.md / CLAUDE.md | README.md |
| 规范 | 小写-连字符.md | document.md |
| 任务 | ticket-{id}.md | ticket-001.md |

## 语言

- 正文：中文
- 代码/注释：英文
- 技术术语：保留原文（API、JSON）
